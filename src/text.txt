## react intro

    What is React?

    1) React is a JavaScript library (not a framework) developed by Facebook (Meta) for building user interfaces (UIs), especially Single Page Applications (SPAs).

    2) It helps developers build reusable UI components that update efficiently when data changes.

    Instead of reloading a whole webpage when something changes, React updates only the parts of the UI that need to change.

    -------
    Why do we need React?

	It makes UI development faster, cleaner, and more efficient.

	Provides a better user experience with smooth updates without page reloads.

## useState Hook

    useState is a react hook that allows you to add state (data that can change over time)
    to a functional component.

    Normally ,in js function , variables disappear after the function runs.
    but with useState,react remembers values b/w  re-render.

    syntax:

        const[state,setstate]=useState(initialValue)

        state---        the current value of the state variable
        setstate---     A function that update thee state
        initialValue--- the starting value of the state variable

## useEffect()


    In react,useEffect is a hook that let you perform side effect in a functional component.
    Side effecct are task that happend outside the normal reacct rendering process Like:

            fetching dta from API
            Updating the browser DOM manually
            starting/stoping timers.

    
    syntax:

        useEffect(()=>{
            //code
        },[dependency 1,dependency 2....])

        []---if we mension the dependency as empty means it will evecute 
                the useEffect  only once at the begining.
